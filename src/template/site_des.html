Welcome to sample blog!

This sample blog show you how to use oceanclouds api and back services to build an web application.

This application use standard user authorize back service, weibo user authorize back service, facebook back service to register and login.

This application use data persistence back service to store blog data.

You can use chrome developer tools or firefox firedebug to watch the data exchange between your browser and back service.

This application is a github static page.
